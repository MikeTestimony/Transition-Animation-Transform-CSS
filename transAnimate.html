CSS Transform, Transition & Animation

Complete Guide with Interactive Examples

1. CSS TRANSFORM

Transform changes the shape, size, and position of elements WITHOUT affecting document flow.

Basic Transform Functions:

TRANSLATE
Hover to move ‚Üí
SCALE
Hover to grow
ROTATE
Hover to spin
SKEW
Hover to slant
MULTIPLE
Hover for combo
3D TRANSFORM
Hover to flip
/* TRANSLATE - Move element */ transform: translateX(50px); /* Move right 50px */ transform: translateY(-30px); /* Move up 30px */ transform: translate(50px, -30px); /* Move right 50px, up 30px */ /* SCALE - Resize element */ transform: scale(1.5); /* 150% size */ transform: scaleX(2); /* Double width */ transform: scaleY(0.5); /* Half height */ /* ROTATE - Spin element */ transform: rotate(45deg); /* Rotate 45 degrees */ transform: rotateX(180deg); /* Flip horizontal */ transform: rotateY(180deg); /* Flip vertical */ /* SKEW - Slant element */ transform: skew(20deg, 10deg); /* Slant X and Y */ /* MULTIPLE TRANSFORMS */ transform: translateX(50px) rotate(45deg) scale(1.2);
Function Description Example
translate(x, y) Move element by x and y pixels translate(50px, 100px)
scale(x, y) Resize element by x and y factor scale(1.5, 2)
rotate(angle) Rotate element by angle rotate(45deg)
skew(x-angle, y-angle) Slant element along axes skew(30deg, 20deg)

2. CSS TRANSITION

Transition smoothly animates property changes over time when triggered (hover, click, etc.)

BASIC
Hover to change color
MULTIPLE
Hover for combo
DELAY
Hover (wait 0.2s)
TIMING
Hover for slow move
/* TRANSITION SYNTAX */ transition: property duration timing-function delay; /* EXAMPLES */ transition: background-color 0.3s ease; transition: transform 0.5s ease-in-out 0.1s; transition: all 0.3s cubic-bezier(0.25, 0.46, 0.45, 0.94); /* INDIVIDUAL PROPERTIES */ transition-property: transform; /* What to animate */ transition-duration: 0.5s; /* How long */ transition-timing-function: ease; /* How it moves */ transition-delay: 0.2s; /* Wait before starting */

Timing Functions:

Timing Function Description Movement Style
ease Slow start, fast middle, slow end Natural feeling
linear Constant speed throughout Mechanical feeling
ease-in Slow start, fast end Accelerating
ease-out Fast start, slow end Decelerating
ease-in-out Slow start and end Smooth acceleration/deceleration

3. CSS ANIMATION

Animation runs automatically using keyframes to define multiple stages of movement.

BOUNCE
Infinite bouncing
PULSE
Infinite pulsing
ROTATE
Infinite spinning
SLIDE IN
Slides from left
COMPLEX
Multi-stage animation
/* ANIMATION SYNTAX */ animation: name duration timing-function delay iteration-count direction fill-mode; /* EXAMPLES */ animation: bounce 2s ease-in-out infinite; animation: slideIn 0.5s ease-out forwards; /* INDIVIDUAL PROPERTIES */ animation-name: bounce; /* Keyframe name */ animation-duration: 2s; /* How long one cycle */ animation-timing-function: ease; /* How it moves */ animation-delay: 0.5s; /* Wait before starting */ animation-iteration-count: infinite; /* How many times */ animation-direction: alternate; /* Forward/backward */ animation-fill-mode: forwards; /* End state */ /* KEYFRAMES */ @keyframes bounce { 0% { transform: translateY(0); } 50% { transform: translateY(-50px); } 100% { transform: translateY(0); } }

Animation Properties:

Property Values Description
animation-iteration-count infinite, 1, 2, 3... How many times to repeat
animation-direction normal, reverse, alternate Which direction to play
animation-fill-mode none, forwards, backwards, both What happens before/after
animation-play-state running, paused Control playback

4. WHEN TO USE EACH

Technique When to Use Trigger Control
Transform Static position/size changes CSS rules (hover, active) Simple on/off
Transition Smooth property changes User interaction (hover, click) Automatic when triggered
Animation Complex, automatic animations Page load, always running Full control with keyframes

5. COMPLETE EXAMPLE - BUTTON

.modern-button { /* Base styles */ background: linear-gradient(135deg, #667eea, #764ba2); color: white; padding: 15px 30px; border: none; border-radius: 50px; cursor: pointer; /* Transform for initial state */ transform: translateY(0); /* Transition for smooth changes */ transition: all 0.3s cubic-bezier(0.25, 0.46, 0.45, 0.94); } .modern-button:hover { /* Transform on hover */ transform: translateY(-5px) scale(1.05); box-shadow: 0 10px 25px rgba(0,0,0,0.2); } .modern-button:active { /* Transform when clicked */ transform: translateY(-2px) scale(1.02); } /* Loading animation */ @keyframes spin { to { transform: rotate(360deg); } } .modern-button.loading::before { content: ''; width: 16px; height: 16px; border: 2px solid transparent; border-top: 2px solid white; border-radius: 50%; animation: spin 1s linear infinite; margin-right: 10px; }
